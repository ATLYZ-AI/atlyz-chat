Installation guide

Add your chatbot in 2 minutes

You don't need to know how to code. Pick the platform your website is built on below and follow the simple steps. It's just copy and paste.

Your embed code
<script src="https://atlyz.com/widget.js?id=your_business_id"></script>

This is the one line you'll paste. You'll find your personal code (with your business ID filled in) inside your Atlyz dashboard on the Install tab.

Which platform is your website built on?

WordPress

~2 min
1

In your WordPress admin, go to Plugins → Add New and install the free plugin "WPCode" (or "Insert Headers and Footers"). Click Activate.

2

Open Code Snippets → Header & Footer from the left menu.

3

Paste your Atlyz code into the Footer box.

The footer box loads the code on every page — exactly what you want.

4

Click Save. Visit your website — the chat button appears in the corner.

💡

No plugin allowed? Go to Appearance → Theme File Editor, open footer.php, and paste the code right before </body>.

Wix

~2 min
1

In your Wix dashboard, go to Settings → Custom Code (under "Advanced").

2

Click + Add Custom Code in the top right.

3

Paste your Atlyz code into the box. Under "Add Code to Pages" choose All pages, and set placement to Body — end.

4

Click Apply. Your chatbot is now live on every page.

💡

Custom Code is available on Wix sites with a paid plan and a connected domain. On the free plan, email us and we'll help you find an alternative.

Squarespace

~1 min
1

From your Squarespace dashboard, go to Settings → Advanced → Code Injection.

2

Paste your Atlyz code into the Footer box.

3

Click Save. Done — the chat button shows on your whole site.

💡

Code Injection requires a Business plan or higher on Squarespace.

Shopify

~3 min
1

In Shopify admin, go to Online Store → Themes.

2

On your current theme, click the ⋯ button and choose Edit code.

3

In the Layout folder, open theme.liquid.

4

Scroll to the bottom, paste your Atlyz code just before </body>, and click Save.

💡

Tip: it's safest to duplicate your theme first (⋯ → Duplicate) so you always have a backup.

Webflow

~2 min
1

Open your Project Settings and go to the Custom Code tab.

2

Paste your Atlyz code into the Footer Code box (labelled "Before </body> tag").

3

Click Save Changes, then Publish your site.

💡

Custom code only appears on the published site, not the Designer preview. Always hit Publish.

Plain HTML website

~1 min
1

Open your website's HTML file (often index.html) in any editor.

2

Find the closing </body> tag near the bottom.

3

Paste your Atlyz code on the line just before it, then save and upload the file to your server.

If your site has multiple pages, add the line to each page (or to your shared footer/template).

Other platform / Not sure

1

Almost every website platform has a setting called "Custom Code", "Header & Footer", "Code Injection" or "Embed". Look for it in your settings.

2

Paste your Atlyz code there and choose to load it on all pages, before the body ends. Save and publish.

3

Can't find it? We'll add it for you, free. See below.
